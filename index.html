Settings

Gameplay
When on: expressions will be generated with sentences containing common punctuation characters and corresponding capitilization. When off: expressions will not contain any punctuation characters or capitilization, however punctuation from enabled phrase lists will still be added. Recommended to be disabled for phrases containing programming lists.
When on: adds a visual keyboard underneath the input area that shows which keys are currently pressed. Not recommended on mobile or low spec devices.
When off: will prevent any WPM, accuracy, and participation statistics from being saved.
